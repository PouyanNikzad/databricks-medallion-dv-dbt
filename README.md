# databricks-medallion-dv-dbt
Designed a scalable Medallion Architecture on Databricks with batch and real‑time ingestion using Delta Lake. Implemented Data Vault 2.0 in Silver for change resilience and auditability, and OBT Gold models for analytics and ML. Used dbt for contracts, data quality, and CI/CD, MLflow for MLOps, and Grafana for monitoring.
